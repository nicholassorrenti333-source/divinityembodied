# Privacy Policy for DivinityFactory

**Effective Date:** February 7, 2026

DivinityFactory is a personal automation application designed to streamline content creation for the Pinterest account @divinityembodied. 

### 1. Data Collection
DivinityFactory does not collect, store, or share any personal data from Pinterest users. The application only accesses the account information provided by the account owner via a secure API Access Token.

### 2. Use of Data
The application uses the Pinterest API solely to:
* Create and manage Pins on the owner's boards.
* Read the owner's board list for content organization.
* Apply SEO metadata to uploaded content.

### 3. Third-Party Sharing
No data accessed through the Pinterest API is ever shared with, sold to, or stored by third-party services. All processing happens locally on the owner's hardware.

### 4. Data Security
Access tokens are stored locally and are never transmitted to any external servers other than Pinterest's official API endpoints.

### 5. Contact
For any questions regarding this personal automation tool, please contact the account owner at divinityembodied [at] yourdomain.com.
